\# Custom Shopify Theme



Custom Shopify storefront theme built with Liquid, schema-driven sections, AJAX cart enhancements, reusable snippets, and metafield-powered merchandising.



\## Features



\- Theme settings schema

\- Reusable homepage sections

\- Product recommendation blocks

\- Newsletter signup

\- SEO snippets

\- AJAX cart drawer

\- Merchant configurable settings
