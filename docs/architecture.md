\# System Architecture



\## Services



\### Shopify Layer

\- checkout

\- collections

\- product data

\- webhooks



\### Next.js Storefront

Responsible for:

\- SSR / ISR rendering

\- SEO metadata

\- storefront routing

\- CMS rendering



\### WordPress CMS

Responsible for:

\- editorial content

\- blog publishing

\- REST delivery



\### Webhook Service

Responsible for:

\- order event ingestion

\- email automation

\- event validation
